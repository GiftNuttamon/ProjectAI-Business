# 📊 ChatBuddy Insights - Admin Dashboard

Dashboard สำหรับแอดมินและเจ้าของร้านค้าออนไลน์ เพื่อดูภาพรวมธุรกิจ ตรวจสอบออเดอร์ สถานะการชำระเงิน และความพึงพอใจของลูกค้าแบบ Real-time โดยดึงข้อมูลตรงจากฐานข้อมูล Google Sheets

## ✨ ฟีเจอร์หลัก (Key Features)
1. **Executive KPIs:** สรุปตัวเลขสำคัญที่เจ้าของธุรกิจต้องรู้ทันที (ยอดขาย, ออเดอร์, จำนวนลูกค้า, สินค้าใกล้หมดสต็อก, ข้อร้องเรียนที่ต้องรีบแก้)
2. **Data Visualization:**
   * **Trend Chart:** กราฟเส้นแสดงแนวโน้มออเดอร์รายวัน
   * **Sentiment Analysis:** กราฟโดนัทแสดงสัดส่วนความพึงพอใจของลูกค้า (Positive, Neutral, Negative)
3. **Smart Data Tables (ระบบแท็บ):**
   * **Order:** แสดงออเดอร์เรียงจากล่าสุด พร้อม Badge สีบอกสถานะ
   *  **Payments:** แสดงข้อมูลการโอนเงิน พร้อม**ระบบไฮไลท์สีแดงแจ้งเตือนสลิปซ้ำ** 
   *  **Products:** สรุปรายการสินค้า พร้อมตัวแดงเตือนหากสต็อก < 10 
   *  **Inquiries & Logs:** รวมประวัติคำถาม, ข้อร้องเรียน และ Log แจ้งเตือนของระบบไว้ในที่เดียว เรียงตามเวลา 

## 🛠 Tech Stack
* **HTML5 / CSS3**
* **Bootstrap 5:** สำหรับจัดการ Layout แบบ Responsive (รองรับมือถือและคอมพิวเตอร์) และระบบ Tabs
* **Chart.js:** สำหรับสร้างกราฟที่สวยงามและมี Interactive
* **PapaParse:** ไลบรารีสำหรับแปลงข้อมูล CSV จาก Google Sheets ให้เป็น JSON Array อย่างรวดเร็ว
* **Font Awesome:** สำหรับไอคอนประกอบให้ UI ดูอ่านง่ายขึ้น
